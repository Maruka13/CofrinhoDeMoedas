Este é um projeto desenvolvido em **Java** que simula o funcionamento de um **cofrinho digital multi-moedas**, com foco na aplicação prática de **Programação Orientada a Objetos (POO)**.

---

## 🚀 Funcionalidades

O sistema permite gerenciar um inventário de moedas com as seguintes operações:

* **Adicionar:** Insere moedas de diferentes valores e nacionalidades (Real, Dólar e Euro)
* **Remover:** Remove moedas específicas do cofrinho
* **Listar:** Exibe todas as moedas armazenadas
* **Totalizador:** Calcula o valor total convertido para Real com base em taxas de conversão

---

## 🧠 Pilares de POO Aplicados

Para garantir um código escalável e de fácil manutenção:

* **Herança:** Classes específicas de moedas herdam de uma classe base `Moeda`
* **Polimorfismo:** O cofrinho manipula uma lista genérica, chamando comportamentos específicos em tempo de execução
* **Encapsulamento:** Proteção dos dados e regras de negócio dentro das classes

## ⚙️ Como executar o projeto

### 1. Pré-requisitos

* Ter o **JDK 17** ou superior instalado

---

### 2. Clone o repositório

```bash
git clone https://github.com/Maruka13/CofrinhoDeMoedas.git
```

---

### 3. Executar o projeto

* Importe o projeto na sua IDE (Eclipse, IntelliJ ou VS Code)
* Execute a classe:

```
Principal.java
```

* O sistema será iniciado via **menu interativo no console**
